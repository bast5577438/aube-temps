Configuration Firebase requise

Le fichier index.html n'a pas encore sa config Firebase. Ouvre le fichier, repère const firebaseConfig en haut du script, et colle les clés de ton projet Firebase (Project settings → Vos applications → SDK config).

Pense aussi à activer Firestore et l'authentification Anonyme dans la console Firebase. Voir le README du dossier pour les étapes complètes.
